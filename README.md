# 🐾 Quadruped Robot Leg Design

> Designing, analyzing, and validating a servo-driven robotic leg through a complete engineering workflow.

---

## 📌 Overview
This project focuses on building a **single leg of a quadruped robot** using theory, calculations, and experiments.

---

## 🎯 Objectives
- Understand torque and its limitations  
- Convert motor specs into real performance  
- Design a functional leg  
- Validate using experiments  

---

## ⚙️ Motor Details
TowerPro MG996R

| Voltage | Torque |
|--------|--------|
| 6V     | 13 kg·cm |
| 4.8V   | 9.4 kg·cm |
